## Beginner Game Design Pathway

For beginners, the recommended pathway is:

1. Read the beginner blog guides.
2. Download the Free Game Design Starter Kit.
3. Download the Free GDD Template.
4. Join The Game Beyond membership.
5. Build a first game concept or prototype.
6. Continue into serious game design methods.
